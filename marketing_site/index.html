The Engine

Every data change — verified.

A cryptographic audit layer that turns every mutation into an immutable trust record.

01🔐
SHA-256 Anchoring

Every payload deterministically hashed — tamper-evident by design. Canonical JSON before hashing.

02⚡
Semantic Delta Engine

>50% numeric change on any field triggers HIGH RISK flag. Full context and delta % returned.

03🎯
Absolute Threshold

Any numeric value >90 auto-flagged instantly, no history needed. Works on any field name.

04📡
God View Dashboard

Live pulse graph, incident feed, Trust Score — real-time, always on. Zero setup required.

05🔑
API Key Management

Create, name, revoke and manage keys for every client app — right from the dashboard.

Universal Integration

// JavaScript / Node.js
await fetch('http://localhost:5000/api/anchor', {
  method: 'POST',
  headers: {
    'x-api-key': 'YOUR_GZ_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    app_id: 'my-app', user_id: 'user_1',
    entity_id: 'employee-42',
    data_payload: { salary: 75000 },
    timestamp: new Date().toISOString()
  })
});

# Python
import requests, datetime

requests.post(
  'http://localhost:5000/api/anchor',
  headers={'x-api-key': 'YOUR_GZ_KEY'},
  json={
    'app_id': 'my-app', 'user_id': 'user_1',
    'entity_id': 'employee-42',
    'data_payload': {'salary': 75000},
    'timestamp': datetime.datetime.utcnow().isoformat() + 'Z'
  }
)
// PHP
$ch = curl_init('http://localhost:5000/api/anchor');
curl_setopt_array($ch, [
  CURLOPT_POST => true,
  CURLOPT_RETURNTRANSFER => true,
  CURLOPT_HTTPHEADER => [
    'x-api-key: YOUR_GZ_KEY',
    'Content-Type: application/json'
  ],
  CURLOPT_POSTFIELDS => json_encode([
    'app_id' => 'my-app',
    'user_id' => 'user_1',
    'entity_id' => 'employee-42',
    'data_payload' => ['salary' => 75000],
    'timestamp' => gmdate('Y-m-d\TH:i:s\Z')
  ])
]);
$response = json_decode(curl_exec($ch), true);
// Java (HttpClient — Java 11+)
var body = """

var res = await client.PostAsJsonAsync(

# Ruby
